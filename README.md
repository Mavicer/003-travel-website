# Wanderful — 003 Travel Website

Cinematic hero section for the "Wanderful" travel brand, built with React + TypeScript + Vite + Tailwind CSS + GSAP.

## Stack

- **Framework:** React 19 + TypeScript
- **Build:** Vite
- **Styling:** Tailwind CSS 3
- **Animation:** GSAP
- **Icons:** lucide-react
- **Fonts:** Inter, Barlow, Instrument Serif, Dirtyline

## Getting Started

```sh
npm install
npm run dev
```

## Switching to Video Background

Set `USE_VIDEO = true` in `src/Hero.tsx` and place your `hero.mp4` in `public/`.
